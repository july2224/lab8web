# lab8web
¿Cuál es la diferencia entre autenticación y autorización?
R: La autenticación verifica la identidad del usuario, mientras que la autorización define los permisos y lo que puede hacer dentro del sistema.

¿Cuál es la función del token JWT en la guía?
R: El token JWT permite mantener la sesión del usuario autenticado y validar su acceso en cada solicitud al backend.
